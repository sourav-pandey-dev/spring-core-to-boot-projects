# 🚀 Spring Core to Boot Projects

This repository documents my structured journey of mastering the **Spring Framework**, starting from core concepts (IoC, DI, XML configuration) to building real-world applications using **Spring Boot, REST APIs, and JPA**.

---

## 📌 Objective

To build a **strong, practical foundation in Spring** by implementing concepts through hands-on projects instead of passive learning.

---

## 🧠 What This Repo Covers

* Spring Core (IoC, DI, Bean Management)
* XML Configuration & Annotation-based Configuration
* Spring Boot Fundamentals
* REST API Development
* Layered Architecture (Controller, Service, Repository)
* Spring Data JPA & H2 Database
* Exception Handling
* Real-world Backend Development

---
